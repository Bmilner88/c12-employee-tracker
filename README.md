# Employee Tracker


## Description

An employee tracking application using MySQL to store and return data through inquirer prompts.
- Built With:
    * JavaScript
    * Node.js
    * MySQL

## Table of Contents

- [Installation](#installation)
- [Testing](#testing)

## Installation

~~~
npm install
~~~


## Testing

~~~
node app
~~~

You must have Node.js AND mysql installed to run the program!
